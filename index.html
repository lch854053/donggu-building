- 데이터
 · 주소 검색 : 행정안전부 도로명주소 API (실시간 갱신)
 · 건축물대장 : 국토교통부 건축물대장 API (매월 갱신, 세움터 기준)
 · 지번 및 좌표 보정 : 국토교통부 V-World API (실시간 갱신)
 · 항공영상 및 필지경계 : 국토교통부 V-World API
